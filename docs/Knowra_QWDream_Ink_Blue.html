Vision Transformer

Vision Transformer 将图像划分为固定大小的 patch，并把每个 patch 映射成向量序列， 从而让 Transformer 可以像处理文本 token 一样处理图像。

核心思路

一张图像被切分为若干个大小相同的 patch。每个 patch 展平后经过线性投影， 得到 patch embedding，再加上位置编码，并与一个可学习的 [CLS] token 一起输入 Transformer Encoder。

ViT 的关键不是“把 Transformer 用在图像上”，而是先把图像重新表达成序列。

需要掌握的内容

重点理解 patch embedding、位置编码、CLS token，以及自注意力在图像任务中的作用。 同时还要能够比较 ViT 与 CNN 在归纳偏置和数据规模要求上的差异。
