# 正式前端迁移记录

## 迁移基线

- 迁移分支：`codex/ui-migration-production`
- 冻结设计源：`docs/前端重构/新UI测试demo/`
- 正式实现：`apps/web/` 下的 vanilla JavaScript SPA
- 迁移原则：复用 Demo 的视觉语言，不复制 Demo 的演示数据和临时 React 结构；正式前端继续使用现有 state、controller、API、Milkdown 编辑器与业务模块。

## 本轮已迁移

1. 统一左侧导航栏：品牌入口、资料库目录、资料范围、业务模块入口和设置入口进入同一套结构。
2. 新增正式资料索引页：基于真实笔记、目录、标签和回收站状态渲染，支持范围切换、搜索、选择、打开、新建和恢复。
3. 重构索引右侧详情栏：保留大块“打开资料”按钮，资料信息与标签常开，关联笔记、内容大纲和附件按需展开。
4. 重构正式编辑工作区：接入冻结稿的文档头、标签页、文件菜单、常用格式工具和连续正文布局，并继续承载原 Milkdown 编辑能力。
5. 重构资料边注：保留双语标题和四个页签，信息与标签常开，目录继续使用 H1/H2/H3 层级；重点、AI、附件及关联能力仍沿用原有控制器和数据链路。
6. 补充索引页与编辑页的窄桌面适配；右栏可收起、可恢复，1280px 与 1440px 视口均无横向溢出。

## 第二轮正式前端收口

1. 索引范围调整为“全部条目 / 最近 / 已收藏 / 回收站”，与左侧资料范围保持一一对应；“最近”按更新时间选取最近 5 条有效资料。
2. 类型、状态和时间由静态展示改为真实筛选器，支持来源类型、整理状态以及创建/编辑时间的正倒序组合；空结果可一次清空全部筛选状态。
3. 顶部搜索面板改为与索引一致的编辑网格结构，标签继续复用蓝色描边组件，不再使用旧版圆角浮层与彩色圆点。
4. 索引卡片原有无业务含义的 A1/B1 标记替换为预计阅读分钟数，并补充资料来源类型；卡片支持单击预览右侧详情、双击直接打开资料。
5. 文档头与正文合并到同一滚动根节点，长文滚动时标题自然离开视口；编辑器聚焦不再给整块正文加蓝色外框。
6. 引用块、行内代码、代码块、分隔线、表格、链接、图片工具栏、源码编辑器、查找替换与表格弹窗统一到当前纸张、蓝线和直角控件体系。
7. 清理源码预览中遗留的圆角目录胶囊，并修复打开源码编辑器时保存状态缺少日期格式化依赖而触发的运行时异常。

## 第三轮浏览器审阅收口

1. 将过小的辅助字号整体上调 1～2px，保持正文 16px 阅读基线不变；资料信息、目录、筛选、状态、时间和菜单文案在 1280px 浏览器视口下更易读。
2. 品牌区继续沿用冻结稿的蓝色 `K` 字标，但改用正式展示字体与更稳定的留白；资料库 `01`、条目编号和阅读分钟数统一为加粗窄体数字。
3. 修复筛选箭头缺少 SVG 尺寸/描边约束的问题；展开态只旋转同一枚 chevron，不再出现浏览器默认的大块黑色填充。
4. 文档标签页高度收窄为 44px，最大宽度收窄为 168px，关闭按钮固定在每个标签页最右侧；空状态和可访问名称改为中文。
5. 三点入口按钮展开时使用蓝色正方形点击态；左侧范围菜单、目录右键菜单、标签页菜单、顶部菜单与编辑器右键菜单统一为直角、蓝色顶线和无毛玻璃的纸张面板。
6. 重要内容标注范围解析增加文档边界校验，删除已标注文本后不再触发 ProseMirror `child.nodeSize` 异常；失效标注不生成 decoration，仍可在唯一文本再次出现时恢复定位。
7. 新增代码块点击恢复插件。真实浏览器已验证代码块可点击获得光标、输入字符并撤销，不再因失效标注事务或浏览器选区映射失败而“无法输入”。
8. 重要文本外观改为蓝色轻底与蓝色底线，悬浮时用酸性黄强调，继续复用正式主题令牌。

## 第四轮交互与密度收口

1. 资料库模块编号恢复为早期细窄字重，条目编号继续保留用于索引定位的加粗层级。
2. 资料索引接入真实分页，默认每页 10 条，并提供 5/10/20/50 条容量选择；页码、容量、总条数与条目序号保持同步。
3. 修复常用格式按钮位于菜单容器内时事件被截断的问题；H1/H2/H3、粗体、斜体、列表、引用和行内代码重新调用同一套 Milkdown 命令，粗体使用 800 字重与深蓝强调。
4. 索引详情与编辑边注均收窄到和左侧目录一致的宽度；窄栏内的资料信息、标签、附件和展开组继续独立滚动。
5. 补齐资料边注标签编辑事件：展开/收起、输入、选择已有标签、创建新标签、移除标签和回车提交均重新接入原标签 controller。
6. 附件行改为直角编辑网格：引用状态使用蓝色方块，打开按钮使用描边方块，当前引用行使用左侧蓝线而非旧版圆角胶囊。
7. 顶部菜单展开态与常用按钮按下态统一为蓝色正方形；底部状态栏增加保存状态、当前资料、字数、行数、大纲、链接、源码和边注快捷入口。
