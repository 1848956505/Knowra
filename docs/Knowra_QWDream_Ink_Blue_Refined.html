Vision Transformer

通过把图像转化为 patch 序列，Vision Transformer 让标准 Transformer 能够直接处理视觉任务。

1,284 字 8 个知识点 3 个标签 更新于 2026/07/11

核心思路

图像首先被切分为若干固定大小的 patch。每个 patch 展平后通过线性映射得到 patch embedding， 加入位置编码后，这些向量与一个可学习的 [CLS] token 一同输入 Transformer Encoder。

ViT 真正改变的不是视觉模型的外形，而是图像进入模型之前的表达方式。

需要掌握的内容

理解 patch embedding、位置编码、自注意力和 CLS token 的作用，并能够比较 ViT 与 CNN 在归纳偏置、数据规模和迁移能力上的差异。
